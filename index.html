CloudSim Summary

CloudSim is a cloud computing simulation framework
used to model and simulate cloud environments
such as datacenters, hosts, virtual machines (VMs),
and cloudlets. It helps students understand
how cloud infrastructure works without using real cloud servers.
In this lab, CloudSim is used to run Example4 simulation and
analyze resource allocation and task execution in a virtual cloud environment.
